使用帮助

该工具用于实时预览 Jinja2 模板与 JSON 上下文的渲染结果。

使用步骤：在左侧输入模板，在中间输入 JSON；可选择预制样式与类型，勾选“应用适配器”（可选）；内容会自动渲染到右侧预览区；错误会显示在预览区下方。

可点击“复制渲染代码”将当前渲染出的 HTML 复制到剪贴板。

示例1：基础渲染与循环
<h2>Hello, {{ name }}!</h2>
<ul>
{% for i in items %}
  <li>{{ loop.index }}. {{ i|upper }}</li>
{% endfor %}
</ul>
                    
{ "name": "Alice", "items": ["apple", "banana", "pear"] }
示例2：条件 + 主题变量 + join
{% if items %}
<div style="border-left:4px solid #faad14; background:#fffbe6; padding:12px;">
  <strong>{{ theme.icon }} 列表</strong>
  <p>共 {{ items|length }} 项：{{ items|join(', ') }}</p>
</div>
{% else %}
<p>暂无数据</p>
{% endif %}
                    
{ "items": ["A", "B", "C"] }
