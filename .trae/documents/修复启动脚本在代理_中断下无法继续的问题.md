## 结论与现状
- 问题不是您的操作：截图显示 `pip` 在网络/代理环境下抛 `InvalidProxyURL`，您按下 `Ctrl+C` 后出现 Windows 批处理提示 “终止批处理操作吗 (Y/N)?”。这是默认行为，和选择 Y/N 无关的根因是安装流程被阻塞。

## 快速临时绕过（不改代码）
- 在命令行手动执行：
  - `python -m pip install -r requirements.txt -i https://pypi.org/simple`
  - 若设置了系统代理，先清空本次会话的代理：`set HTTP_PROXY=`、`set HTTPS_PROXY=`，再执行上面的安装命令。
- 如果依赖已安装，可直接运行：`python app.py`
