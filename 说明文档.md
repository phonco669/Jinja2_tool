# Jinja2 实时渲染工具项目说明文档

## 1. 项目规划
### 1.1 项目目标
开发一个在 Windows 环境下可一键启动的轻量级工具，允许用户输入 Jinja2 模板代码（及可选的上下文数据），实时预览渲染后的 HTML 效果。

### 1.2 技术选型
- **编程语言**: Python
- **Web 框架**: Flask (轻量级，易于实现 API 和静态服务)
- **模板引擎**: Jinja2
- **前端界面**: HTML5 + JavaScript (Fetch API) + CSS (Flexbox 布局)
- **启动方式**: Windows 批处理脚本 (.bat) 实现一键安装依赖并启动

### 1.3 功能需求
1. **代码编辑区**: 用户输入 Jinja2 模板代码。
2. **数据编辑区**: 用户输入 JSON 格式的上下文数据（可选）。
3. **实时预览区**: 展示渲染后的 HTML 结果。
4. **实时性**: 输入变动时自动触发渲染（或提供渲染按钮）。
5. **错误处理**: 若模板或数据格式错误，需在界面友好提示。

## 2. 实施方案
### 2.1 目录结构
```
Jinja2_tool/
├── app.py              # Flask 主程序
├── requirements.txt    # 项目依赖
├── start.bat           # Windows 启动脚本
├── README.md           # 使用说明
└── templates/
    └── index.html      # 前端界面
```

### 2.2 核心模块设计
- **后端 (`app.py`)**:
    - `/`: 返回主界面。
    - `/render`: POST 接口，接收 `template` 和 `data`，返回渲染结果或错误信息。
- **前端 (`index.html`)**:
    - 左右分栏布局（或左中右）。
    - 监听输入框 `input` 事件，防抖动（debounce）后发送请求给后端。
    - 更新预览区内容。
